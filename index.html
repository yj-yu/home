SW/HW codesign project

with Gangjae Jang, Jaemin Kim, Naehyuck Chang

Project "CodeWar" was developed using the PowerPC & MicroBlaze Virtex-5 Embedded Development Kit and bluetooth embedded module (FB755AC) for multiplay game.
In the recommendation of the professor, I developed the final term project and submitted it to the creative engineering design competition. The user must provide the analog signal of the Morse code controller as an input to support bombardment by unlocking all the given secret code faster than the competitor.
The Codewar game supports up to 2 multi-players and is designed to improve the morse code understanding of soldiers.
